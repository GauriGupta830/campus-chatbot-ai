## Explain in brief what all you did?
We built SVIMS CampusBot — a fully functional AI-powered 
campus information assistant for Shri Vaishnav Institute 
of Management & Science, Indore.

The chatbot helps students, freshers and visitors get 
instant answers about courses, admissions, fees, faculty, 
placements, clubs, events and campus facilities — all 
through natural conversation in Hindi or English.

Key work done:
- Scraped entire SVIMS website using BeautifulSoup
- Built vector knowledge base using FAISS + LangChain
- Integrated Groq AI (Llama 3) for fast, accurate responses
- Designed professional UI with custom CSS in Streamlit
- Added Contact Directory, Events Calendar and Campus Map
- Deployed live on Streamlit Cloud
